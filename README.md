# 🤖 JARVIS AI Voice Assistant

An intelligent **voice-activated AI assistant** inspired by JARVIS from Iron Man.
This project demonstrates **speech recognition, natural language processing, and interactive UI** to create a natural human-computer interaction system.

---

## 📌 Overview

JARVIS AI Voice Assistant allows users to interact with a computer using **voice commands**.
The assistant listens to user input, processes the command using NLP techniques, and generates responses through **text or speech output**.

This project was developed as a **learning project for AI, web development, and human-computer interaction**.

---

## 🚀 Features

* 🎙 **Voice Command Recognition** – Detects and processes spoken commands
* 🧠 **Natural Language Processing (NLP)** – Understands user intent
* 💬 **Interactive Chat Interface** – Displays conversation between user and AI
* 🔊 **Speech Response System** – Provides audio feedback
* 📊 **System Status Panel** – Displays system activity and status
* 💾 **Conversation Memory** – Stores previous interactions
* 🖥 **Modern JARVIS-Style UI** – Futuristic interface inspired by Iron Man

---

## 🛠 Technologies Used

* **Python** – Backend logic
* **Flask** – Web server framework
* **HTML5** – Structure of the interface
* **CSS3** – Styling and UI design
* **JavaScript** – Interactive functionality
* **Speech Recognition / NLP libraries**

## ⚙ Installation

1️⃣ Clone the repository

```
git clone https://github.com/yourusername/JARVIS-AI-Voice-Assistant.git
```

2️⃣ Navigate to the project folder

```
cd JARVIS-AI-Voice-Assistant
```

3️⃣ Install required dependencies

```
pip install -r requirements.txt
```

4️⃣ Run the application

```
Terminal - 1 : python agent.py dev
Terminal - 2 : python app.py

```

5️⃣ Open the browser and go to

```
http://127.0.0.1:5000/
```

---

## 💡 Usage

1. Start the application.
2. Click **Start Listening**.
3. Speak commands to interact with the assistant.
4. The assistant will process the command and respond accordingly.
